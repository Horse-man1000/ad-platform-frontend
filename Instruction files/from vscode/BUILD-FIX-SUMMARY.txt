1. PRISMA VERSION MISMATCH (BLOCKER)

   - db.sqlite committed to repo at frontend/prisma/db.sqlite

4. MISSING NODE RUNTIME DECLARATION
   - App Router API routes require explicit Node.js runtime when using Prisma
   - Fix: Added `export const runtime = 'nodejs';` to top of both route files:
       app/api/health/route.js
       app/api/clients/route.js

5. @prisma/client IN WRONG DEPENDENCY SECTION
   - npm install --save-dev moved @prisma/client to devDependencies
   - Vercel does not install devDependencies in production
   - Fix: Moved @prisma/client back to dependencies in package.json
